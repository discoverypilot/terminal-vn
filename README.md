# terminalVN
i made a rudimentary visual novel engine that runs in the terminal using only standard C++. included is a short demo of its features, which should run on linux or mac in pretty much any unix shell. powershell/windows is not supported.

## features
- [x] basic text/border functionality
- [x] text is displayed character by character with a delay
- [ ] read ASCII art from .txt files and display in main window
- [ ] full screen ASCII art for CG scenes
- [ ] read script text from .txt files to display dialogue/other text
- [ ] automatically wrap text by full word and delete any empty space at the start of a line
- [ ] display choices to user in a centered list
- [ ] let user type in a choice from the terminal when offered

## how to use
if you want to try out the demo, run the binary file from your unix shell cli of choice. mine is kitty :3
